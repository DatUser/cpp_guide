# C++ Guides

Giving tips for C++:

<ins>WSL/Unix local install:</ins>

Create env:
```
mamba create -n cpp_guide -c conda-forge xeus-cling jupyterlab
```

Activate env:
```
mamba activate cpp_guide
```

Run jupyterlab:
```
jupyter lab
```
